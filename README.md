# 데일리픽

무료 도메인이나 무료 호스팅에 올릴 수 있는 정적 블로그형 광고 사이트입니다.

## 먼저 바꿀 것

1. `script.js`의 `YOUR_SMARTSTORE_URL`을 실제 스마트스토어 주소로 바꾸세요.
2. `contact.html`의 `contact@example.com`을 실제 문의 이메일로 바꾸세요.
3. 모든 HTML 파일의 `ca-pub-0000000000000000`을 본인 Google AdSense 게시자 ID로 바꾸세요.
4. 모든 HTML 파일의 `data-ad-slot="0000000000"`을 발급받은 광고 단위 ID로 바꾸세요.
5. `ads.txt`의 `pub-0000000000000000`을 본인 게시자 ID로 바꾸세요.
6. 사이트 이름을 바꾸려면 모든 HTML 파일의 `데일리픽` 문구를 원하는 이름으로 바꾸세요.

## 무료로 올리는 방법

- GitHub Pages: 저장소에 파일을 올리고 Pages를 켜면 무료 주소가 생깁니다.
- Netlify: 폴더를 드래그해서 배포할 수 있고 무료 서브도메인을 제공합니다.
- Cloudflare Pages: 무료 배포와 무료 서브도메인을 제공합니다.

## 운영 팁

- 광고 심사 전에는 복사한 글보다 직접 작성한 글을 10개 이상 쌓는 편이 좋습니다.
- 기사마다 800자 이상, 구매 기준과 실제 사용 상황을 넣으면 블로그 품질이 좋아집니다.
- 스마트스토어 링크가 들어간 글에는 광고 또는 제휴 안내를 분명히 적어두세요.
- 애드센스 승인은 도메인을 실제로 연결한 뒤 신청해야 합니다.
- 승인 전에는 광고가 빈칸으로 보일 수 있습니다. 정상입니다.
